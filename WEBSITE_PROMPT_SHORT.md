## About MobileDigger

### Elevator Pitch
"Organize music like a DJ - swipe right to like, left to dislike. See waveforms, analyze spectrograms, work completely offline."

### Key Features
1. **Swipe Gestures** - Sort files instantly (right=like, left=dislike)
2. **Real-Time Waveforms** - Visual audio representation, seekable
3. **Spectrograms** - FFT frequency analysis for audio quality checks
4. **All Formats** - MP3, FLAC, AIFF, WAV, M4A, AAC, OGG
5. **Privacy First** - Zero data collection, 100% offline
6. **Smart Miniplayer** - Auto-appears when scrolling
7. **Multi-Select** - Batch organize multiple files
8. **Custom Themes** - 9 beautiful Material 3 themes
9. **Professional Grade** - Handles 10,000+ files smoothly
10. **Open Source** - Transparent, auditable code

### Target Users
- DJs organizing record pools and Beatport downloads
- Music producers sorting samples
- Collectors managing large music libraries
- Anyone with 1000+ audio files to organize
